# 공휴일

홍콩에 본사를 둔 기업으로서, 주로 **중국 본토**와 **홍콩**의 공휴일을 따릅니다. 이 날에는 사무실이 휴무입니다. 정규 업무 시간은 [소개](about-us.md)를 참조해 주세요.

!!! info "공휴일 중 응답 시간"
    공휴일에는 평소보다 응답이 느려질 수 있습니다. 기술 지원 및 고객 지원 팀은 일반적으로 **48시간 이내**에 답변드립니다.

    양해 부탁드립니다. 저희는 고객에게 훌륭한 제품과 서비스를 제공하는 동시에, 팀원들의 건강한 워라밸을 지원하기 위해 노력하고 있습니다. 이해해 주셔서 감사합니다.
