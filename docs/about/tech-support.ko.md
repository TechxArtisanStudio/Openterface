# 기술 지원

예상대로 작동하지 않을 때 좌절스러운 것을 이해하며, **우리가 도와드리겠습니다**. 우리의 문제 해결 양식이 설정과 직면한 문제에 대한 세부사항을 공유할 수 있도록 **단계별로** 안내해드립니다. 이 정보를 통해 문제를 더 잘 이해하고 더 효과적으로 지원할 수 있습니다.

원인을 파악하고 최적의 해결책을 찾기 위해 함께 노력하는 동안 기다려 주시기 바랍니다.

다음은 지원을 받을 수 있는 방법들입니다:

- [FAQ 확인](/faq): 거기서 답을 찾을 수 있을 것입니다.
- [문제 해결 양식](/support): 작성해 주시고 설정을 더 잘 이해할 수 있도록 도와주세요.
- [커뮤니티 참여](/community): 더 즉각적인 채팅을 선호한다면, 여기는 우리 개발팀과 다른 훌륭한 사용자들이 모이는 곳입니다.
- [GitHub Issues](/app): 앱의 GitHub 저장소에서 문제를 확인하고 생성하세요
- 이메일 문의: [support@openterface.com](mailto:support@openterface.com)
