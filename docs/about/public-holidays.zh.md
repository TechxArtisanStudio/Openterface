# 公众假期

作为一家总部位于香港的公司，我们主要遵循**中国内地**和**香港**的公众假期。这些日期我们的办公室将休息。有关我们的常规工作时间，请参阅[关于我们](about-us.md)。

!!! info "假期期间的响应时间"
    在公众假期期间，我们的响应可能比平时较慢。我们的技术与客户支持团队通常会在 **48 小时**内回复。

    敬请理解。我们致力于为客户提供优质的产品和服务，同时也支持团队保持健康的工作与生活平衡。感谢您的耐心。
