## 翻译流程

!!! info "AI 驱动的翻译"
    我们使用**大语言模型 (LLMs)** 来帮助我们组织和翻译多语言文档。

!!! note "质量保证"
    - **英语**作为我们的主要语言和事实来源
    - 我们的团队编写、审查和维护所有翻译
    - 我们使用 AI 工具协助初始翻译，但内容经过彻底的人工审查
    - 如果您在我们的翻译中发现任何错误或不准确之处，请通过 [email](mailto:info@techxartisan.com) 或 [表单](/feedback) 向我们报告。我们感谢您的帮助！
