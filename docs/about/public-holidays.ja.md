# 祝日

香港に拠点を置く企業として、主に**中国本土**と**香港**の祝日に準拠しています。これらの日はオフィスは休業となります。通常の営業時間については[私たちについて](about-us.md)をご覧ください。

!!! info "祝日中の対応時間"
    祝日中の対応は通常より遅くなる場合があります。技術サポートおよびカスタマーサポートチームは通常**48時間以内**にご返信いたします。

    ご理解のほどよろしくお願いいたします。お客様に優れた製品とサービスを提供することに努めるとともに、チームの健全なワークライフバランスをサポートしています。ご了承のほどお願いいたします。

!!! note "振替休日の取り扱い"
    2026年の清明節は日曜日に当たるため、翌日（4月6日）が振替休日として指定されています。この日はイースターマンデーと重なるため、その翌日で祝日でない日（4月7日）が追加の休日として設定されています。  
    2026年の仏誕と重陽節はともに日曜日に当たるため、それぞれの翌日が振替休日として指定されています。
