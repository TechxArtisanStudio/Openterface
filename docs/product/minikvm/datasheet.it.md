# Foglio tecnico Openterface Mini-KVM

## Panoramica

L'Openterface™ Mini-KVM è un dispositivo ricco di funzionalità, a livello industriale, open source e basato sulla comunità. Offre una soluzione leggera e veloce KVM-over-USB, che ti permette di controllare un computer headless (indicato come computer Target) direttamente dal tuo laptop o desktop (indicato come computer Host), tramite una semplice connessione USB e HDMI. Questo approccio compatto elimina la necessità di tastiere, mouse, monitor o qualsiasi configurazione di rete, semplificando la tua installazione e migliorando l'efficienza.

## Connessioni

### Connettività / Interfacce

1. **Porta USB-C (Femmina)** ⓵  
   Porta USB come dispositivo, collegata al computer Host per il trasferimento dati tramite hub USB integrato

2. **Porta USB-C (Femmina)** ②  
   Porta USB come dispositivo, collegata al computer Host per l'output HID del tastierino e del mouse tramite hub USB integrato

3. **Porta di ingresso HDMI (Femmina)** ③  
   Ingresso HDMI da parte del computer Target

4. **Porta USB-A 2.0 commutabile (Femmina)** ④  
   Porta USB come host, utilizzata da parte del computer Host o Target in qualsiasi momento, ma non contemporaneamente

5. **Interruttore a scorrimento** ⑤  
   Per commutare la connessione della porta USB-A 2.0 tra Host e Target

6. **Punti di estensione** ⑥  
   Per ulteriori informazioni, controlla i punti di estensione per l'uso da parte dello sviluppatore.

L'Openterface Mini-KVM è il compagno ideale per una vasta gamma di utenti e scenari:

- Professionisti IT che risolvono problemi sui server
- Tecnici che servono ATM, VLT e kioschi
- Sviluppatori che gestiscono dispositivi di calcolo edge
- Appassionati di tecnologia che sperimentano con computer a scheda singola
- Professionisti che richiedono operazioni locali sicure in ambienti di rete isolati, ad esempio coloro che gestiscono asset criptovaluta
- Chiunque abbia bisogno di workflow integrati frequentemente tra computer personali e lavoro.
