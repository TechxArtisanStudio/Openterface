## Resumen

El Mini-KVM es un dispositivo KVM-over-USB de tamaño de bolsillo que se conecta al equipo objetivo mediante HDMI y USB, y expone el vídeo, audio y dispositivos de entrada del objetivo a un equipo anfitrión (portátil o sobremesa). Está pensado para técnicos de campo, uso en centros de datos y cualquiera que necesite una consola portátil para sistemas sin pantalla o integrados.

Puntos clave:

- Formato portátil y compacto para solución de problemas en movimiento.
- Experiencia plug-and-play: conexión HDMI + USB al anfitrión.
- Software y firmware de código abierto para facilitar contribuciones de la comunidad.
- Captura de vídeo de baja latencia apta para uso interactivo.
