## Visão geral

O Mini-KVM é um dispositivo KVM-over-USB de bolso que se conecta a uma máquina alvo via HDMI e USB e expõe o vídeo, áudio e dispositivos de entrada da máquina ao computador anfitrião (laptop ou desktop). Foi projetado para técnicos de campo, uso em datacenters e quem precisa de um console portátil para sistemas headless ou embarcados.

Pontos principais:

- Formato portátil e compacto para solução de problemas em movimento.
- Experiência plug-and-play: conexão HDMI + USB ao host.
- Software e firmware open-source para permitir contribuições da comunidade.
- Captura de vídeo de baixa latência adequada para uso interativo.
