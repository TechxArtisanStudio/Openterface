## Prezentare generală

Mini-KVM este un dispozitiv KVM-over-USB de dimensiunea buzunarului care se conectează la o mașină țintă prin HDMI și USB și expune video, audio și dispozitivele de intrare ale țintei către un host (laptop sau desktop). Este proiectat pentru tehnicieni de teren, utilizare în centre de date și pentru oricine are nevoie de o consolă portabilă pentru sisteme headless sau incorporate.

Puncte cheie:

- Factor de formă portabil și compact pentru depanare în mișcare.
- Experiență plug-and-play: conexiune HDMI + USB către host.
- Software și firmware open-source pentru a permite contribuțiile comunității.
- Captură video cu latență scăzută, potrivită pentru utilizare interactivă.
