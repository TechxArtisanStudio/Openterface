# Ficha técnica do Openterface Mini-KVM

## Visão geral

O Openterface™ Mini-KVM é um dispositivo rico em recursos, de produção, open-source e movido pela comunidade. Ele oferece uma solução KVM-over-USB leve e rápida, permitindo que você controle um computador headless (chamado de computador Target) diretamente do seu próprio laptop ou desktop (chamado de computador Host), via uma simples conexão USB e HDMI. Esta abordagem compacta elimina a necessidade de teclados, mouses, monitores adicionais ou qualquer configuração de rede, simplificando sua instalação e aumentando a eficiência.

## Conexões

### Conectividade / Interfaces

1. **Porta USB-C (Fêmea)** ⓵  
   Como uma porta de dispositivo USB, conectando-se ao computador Host para transferência de dados via hub USB embutido

2. **Porta USB-C (Fêmea)** ②  
   Como uma porta de dispositivo USB, conectando-se ao computador Host para emular saída HID de teclado e mouse via hub USB embutido

3. **Porta de Entrada HDMI (Fêmea)** ③  
   Fonte de entrada HDMI do computador Target

4. **Porta USB-A 2.0 (Fêmea) com Comutação** ④  
   Como uma porta USB host, utilizada por qualquer um dos computadores Host ou Target em qualquer momento, mas não simultaneamente

5. **Interruptor de Comutação** ⑤  
   Para alternar a conexão da porta USB-A 2.0 entre o Host e o Target

6. **Pinos de Extensão** ⑥  
   Para mais informações, consulte os Pinos de Extensão para uso por desenvolvedores.
