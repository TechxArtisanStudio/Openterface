# Fișa tehnică Openterface Mini-KVM

## Prezentare generală

Openterface™ Mini-KVM este un dispozitiv bogat în funcții, de producție, open-source și comunitar. Oferă o soluție rapidă KVM-over-USB, care vă permite să controlați un computer headless (numit computerul Target) direct de la propriul laptop sau computer desktop (numit computerul Host), prin intermediul unei conexiuni simple USB și HDMI. Această abordare compactă elimină nevoia de taste, mouri, monitoare sau orice configurare rețea, simplificându-vă instalarea și îmbunătățind eficiența.

## Conexiuni

### Conectivitate / Interfețe

1. **Port USB-C (feminin)** ⓵  
   Ca port USB device, conectat la computerul Host pentru transfer de date prin hub-ul USB integrat

2. **Port USB-C (feminin)** ②  
   Ca port USB device, conectat la computerul Host pentru simularea ieșirii HID tastatură și mouse prin hub-ul USB integrat

3. **Port de intrare HDMI (feminin)** ③  
   Sursă HDMI de la computerul Target

4. **Port USB-A 2.0 (feminin) comutabil** ④  
   Ca port USB host, utilizat de către computerul host sau target în orice moment, dar nu simultan

5. **Comutator** ⑤  
   Pentru comutarea conexiunii portului USB-A 2.0 între host și computerul target

6. **Pini de extensie** ⑥  
   Pentru mai multe detalii, consultați Pini de extensie pentru utilizare de dezvoltator.

## Aplicații

Openterface Mini-KVM este companionul perfect pentru o varietate largă de utilizatori și scenarii:

- Profesioniști IT care diagnostichează servere
- Tehnicieni care servicează ATM-uri, VLT-uri și kioscouri
- Dezvoltatori care gestionează dispozitive de calcul la marginea rețelei
- Enthuziaști tehnologici care experimentează cu computere monolitice
- Profesioniști care necesită operații locale sigure în rețelele segmentate, cum ar fi cei care gestionează active criptomonede
- Oricine are nevoie de fluxuri de lucru integrate frecvent între computerele personale și cele de serviciu.
