# **박스 내용물** | 사전 출시 패키지 | Openterface KVM-Go 시리즈

**HDMI**, **DP**, 또는 **VGA** 버전의 세 가지 완전한 툴킷 옵션을 제공합니다. 이들 간의 유일한 차이점은 핵심 제품 자체입니다. 다른 모든 내용물은 세 옵션 모두 동일합니다.

각 툴킷에는 다음이 포함됩니다:

1. **핵심 제품**: Openterface KVM-GO (HDMI, DP 또는 VGA 버전 선택)
2. **툴킷 가방**: 컴팩트한 휴대용 케이스
3. **USB 케이블**: 
   - 0.5m 검은색 Type-C to USB-A 케이블 (Type-C 어댑터 포함)
   - 2.5m 주황색 Type-C to Type-C 케이블 (USB-A 어댑터 포함)
4. **3D 프린팅 캡**: microSD 저장소가 있는 맞춤형 캡
5. **빠른 시작 가이드**
