## 사용 가능한 모델

- **KVM-Go HDMI 수** — 모던 장치용 직접 HDMI 연결
- **KVM-Go DisplayPort 수** — 고성능 DisplayPort 지원  
- **KVM-Go VGA 수** — 레거시 시스템 호환성 (출시 예정)

모든 모델은 동일한 초소형 디자인과 핵심 기능을 공유하며, 내장 비디오 커넥터만 다릅니다.
