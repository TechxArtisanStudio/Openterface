# **Ce Este în Cutie** | Pachete de Pre-Lansare | Seria Openterface KVM-Go

Oferim trei opțiuni complete de kit de instrumente: versiunea **HDMI**, **DP** sau **VGA**. Singura diferență între ele este produsul principal în sine. Toate celelalte conținuturi sunt identice pentru toate cele trei opțiuni.

Fiecare kit de instrumente include:

1. **Produs Principal**: Openterface KVM-GO (alegeți versiunea HDMI, DP sau VGA)
2. **Geantă pentru Instrumente**: Husă compactă de transport
3. **Cabluri USB**: 
   - Cablu Type-C către USB-A negru de 0,5 m cu adaptor Type-C
   - Cablu Type-C către Type-C portocaliu de 2,5 m cu adaptor USB-A
4. **Capac Imprimat 3D**: Capac personalizat cu stocare microSD
5. **Ghid de Pornire Rapidă**
