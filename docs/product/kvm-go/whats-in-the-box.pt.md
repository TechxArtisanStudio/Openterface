# **O Que Está na Caixa** | Pacotes de Pré-Lançamento | Série Openterface KVM-Go

Oferecemos três opções completas de kit de ferramentas: versão **HDMI**, **DP** ou **VGA**. A única diferença entre eles é o produto principal em si. Todos os outros conteúdos são idênticos nas três opções.

Cada kit de ferramentas inclui:

1. **Produto Principal**: Openterface KVM-GO (escolha a versão HDMI, DP ou VGA)
2. **Bolsa de Ferramentas**: Estojo de transporte compacto
3. **Cabos USB**: 
   - Cabo Type-C para USB-A preto de 0,5 m com adaptador Type-C
   - Cabo Type-C para Type-C laranja de 2,5 m com adaptador USB-A
4. **Tampa Impressa em 3D**: Tampa personalizada com armazenamento microSD
5. **Guia de Início Rápido**
