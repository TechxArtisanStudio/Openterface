# **Contenu de la Boîte** | Packages de Prélancement | Openterface KVM-Go Series

Nous proposons trois options de kits complets : version **HDMI**, **DP** ou **VGA**. La seule différence entre eux est le produit principal lui-même. Tous les autres contenus sont identiques pour les trois options.

Chaque kit comprend :

1. **Produit Principal** : Openterface KVM-GO (choisissez la version HDMI, DP ou VGA)
2. **Sacoche de Rangement** : Étui de transport compact
3. **Câbles USB** : 
   - Câble Type-C vers USB-A noir de 0,5 m avec adaptateur Type-C
   - Câble Type-C vers Type-C orange de 2,5 m avec adaptateur USB-A
4. **Capuchon Imprimé en 3D** : Capuchon personnalisé avec stockage microSD
5. **Guide de Démarrage Rapide**
