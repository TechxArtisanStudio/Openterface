# **包装内容** | 预发布套装 | Openterface KVM-Go 系列

我们提供三种完整的工具包选项——**HDMI**、**DP** 或 **VGA** 版本。它们之间唯一的区别是核心产品本身。所有其他内容在三个选项中都是相同的。

每个工具包包含：

1. **核心产品**：Openterface KVM-GO（选择 HDMI、DP 或 VGA 版本）
2. **工具包收纳袋**：紧凑型便携包
3. **USB 数据线**： 
   - 0.5米黑色 Type-C 转 USB-A 数据线，配 Type-C 转接头
   - 2.5米橙色 Type-C 转 Type-C 数据线，配 USB-A 转接头
4. **3D 打印保护帽**：定制保护帽，带 microSD 存储
5. **快速入门指南**
