# 🚀 KVM-Go HDMI Betaバッチ発送中！

![kvm-go-beta-test-1](https://assets.openterface.com/images/kvm-go/beta-test/1.webp){:style="max-width:720px"}

**KVM-GO betaチーム**への参加に登録していただいたすべての皆様に心から感謝します。最初の少量のbetaデバイスバッチを梱包して出荷しました！🎉

このコンパクトな小さなKVMを、超ポータブルでありながら本格的なワークフローを処理できるほど強力に、私たちが想像した通りに動作させるのは、かなりの道のりでした。途中でいくつかの厳しい課題に直面しました（こんなに小さな筐体でより高い解像度をプッシュするとき、熱の問題は現実です）が、皆さんの助けを借りて取り組んでおり、急速に学んでいます。

![kvm-go-beta-test-6](https://assets.openterface.com/images/kvm-go/beta-test/6.webp){:style="max-width:720px"}

私たちのオープンソースハードウェアとソフトウェアの旅を支援してくださるすべての皆様に心から感謝します。🙏 コミュニティのテスト、フィードバックの共有、そして情報の拡散なしには、このプロジェクトは不可能でした。

さらなるBetaテストのラウンドが予定されていますが、枠は限られています。デバイスを限界まで押し上げることが大好きなメーカー、システム管理者、またはITティンカラーの方は、注目していてください。次のイテレーションを形作るために皆さんの助けが必要です。
