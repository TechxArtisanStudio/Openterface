# 介绍Openterface KVM-Go系列 - 预发布公告！

我们很高兴宣布**Openterface KVM-Go系列**的预发布 - 我们的新一代超紧凑型KVM-over-USB解决方案！这款革命性设备通过钥匙扣大小的外形和内置视频连接器，将便携性提升到新的水平，无需单独的线缆。

KVM-Go系列代表着我们广受欢迎的Mini-KVM的巨大进化，为IT专业人士、开发人员和技术爱好者提供增强的性能、最大的便利性和前所未有的便携性，满足他们对工作流程效率的需求。

![KVM-Go系列产品图片](https://assets.openterface.com/images/kvm-go/hdmi-p1.webp)

## 终极便携式KVM解决方案

Openterface KVM-Go系列专为一个目标而设计：**使无头设备管理尽可能便携**。以下是其突出特点：

### 超紧凑设计
- **钥匙扣大小**：18 × 18 × 55毫米（0.71 × 0.71 × 2.17英寸）
- **轻量级**：仅约25克（0.9盎司） - 比大多数钥匙扣更轻
- **内置连接器**：具有HDMI、DisplayPort或VGA公头连接器的直接插入能力

### 增强的性能
- **4K@60Hz视频支持**：从Mini-KVM的1080p@30Hz大幅升级
- **USB 3.0连接**：更快的数据传输和改进的响应速度
- **MicroSD卡槽**：文件传输和可引导镜像创建功能

### 无需线缆管理
- **直接插入**：内置视频连接器消除单独的HDMI/DP/VGA线缆
- **30秒设置**：从开箱到控制目标设备
- **LED指示灯**：显示连接状态的视觉反馈（蓝色=目标，绿色=主机）

![KVM-Go系列功能](https://assets.openterface.com/images/kvm-go/hdmi-p6.webp)

## 三种型号满足各种需求

我们推出三种不同型号，以满足您的所有连接需求：

### KVM-Go HDMI公头
- **最适合**：现代设备、服务器、工作站
- **连接器**：内置HDMI公头连接器
- **兼容性**：HDMI 1.4+设备

### KVM-Go DisplayPort公头  
- **最适合**：高分辨率显示器、专业设置
- **连接器**：内置DisplayPort公头连接器
- **兼容性**：DisplayPort 1.2+设备

### KVM-Go VGA公头（即将推出）
- **最适合**：传统系统、旧服务器
- **连接器**：内置VGA公头连接器
- **兼容性**：标准VGA设备

## Openterface KVM-Go的不同之处

您可能想知道为什么应该在各种可用的KVM解决方案中选择KVM-Go系列？原因如下：

1. **超紧凑且高效**：专为最大便携性而设计，我们的KVM-over-USB解决方案在网络访问有限或没有网络的环境中表现出色，提供真正便携、独立于网络、闪电般快速的故障排除工具。

2. **内置便利性**：通过将视频连接器直接集成到设备中，我们消除了便携式KVM解决方案中最大的痛点之一 - 管理和携带单独的视频线缆。

3. **增强的性能**：凭借4K@60Hz支持和USB 3.0连接，KVM-Go系列在钥匙扣大小的封装中提供专业级性能。

4. **社区驱动和开源**：KVM-Go系列的核心是培育创新和协作的社区。通过拥抱开源价值观，我们邀请用户贡献自定义功能和改进，丰富工具的能力和多功能性。
