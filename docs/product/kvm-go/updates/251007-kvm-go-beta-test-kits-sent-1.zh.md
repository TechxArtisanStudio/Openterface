# 🚀 KVM-Go HDMI Beta批次正在发货！

![kvm-go-beta-test-1](https://assets.openterface.com/images/kvm-go/beta-test/1.webp){:style="max-width:720px"}

非常感谢所有报名加入我们**KVM-GO beta团队**的人。我们刚刚打包并发货了第一小批beta设备！🎉

让这款小巧的KVM按照我们想象的方式工作确实是一段不小的旅程，既要超便携又要足够强大以处理严肃的工作流程。我们在过程中遇到了一些棘手的挑战（在如此小的机身中推动更高分辨率时，发热问题确实存在），但在你们的帮助下，我们正在努力解决并快速学习。

![kvm-go-beta-test-6](https://assets.openterface.com/images/kvm-go/beta-test/6.webp){:style="max-width:720px"}

我们真诚感谢所有支持我们开源硬件和软件之旅的人。🙏 没有社区的测试、分享反馈和传播信息，这个项目是不可能实现的。
