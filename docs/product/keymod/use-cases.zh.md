# 使用场景 | Openterface KeyMod

KeyMod 专为需要快速键盘和鼠标输入但完整键盘不可用或不方便的时刻而设计。将手机用作输入设备——将 KeyMod 插入目标设备即可开始控制。

## 主要使用场景

### **户外显示屏电脑**

无需携带键盘即可控制户外安装的显示屏电脑。KeyMod 将手机连接到显示屏 PC，用于配置、故障排除或内容更新。

### **LED signage 播放器**

在现场管理 LED signage 播放器和控制器。插入 KeyMod，将手机用作键盘，无需寻找键盘即可进行更改。

### **信息亭**

快速配置、调试或维护信息亭。当您需要与信息亭操作系统或软件交互时，KeyMod 提供即时的键盘和鼠标访问。

### **智能电视和机顶盒**

控制有显示屏但用遥控器操作不便的智能电视、机顶盒和媒体播放器。将手机用作完整键盘和触控板。

### **快速工作流快捷方式**

当您需要比在设备之间切换更快的工作流程时，KeyMod 让您可以使用手机进行快速输入。非常适合开发者、IT 人员以及需要偶尔使用键盘但不想携带额外设备的人。

## 为什么选择 KeyMod？

- **便携** — 手机随身携带；KeyMod 紧凑
- **即插即用** — 目标设备无需安装软件
- **双连接** — USB 可靠，蓝牙无线使用
- **开源** — 自定义配置、宏和工作流
