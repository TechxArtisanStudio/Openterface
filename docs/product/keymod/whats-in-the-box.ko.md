# **구성품** | Openterface KeyMod

KeyMod는 현재 출시 전 단계입니다. 패키지 내용과 패키징 디자인은 아직 개발 중이며 배송 전에 변경될 수 있습니다.

## 예상 구성품 *(임시)*

1. **KeyMod 유닛** — USB + 블루투스 HID 에뮬레이터 (2-in-1 또는 USB C 버전 선택)
2. **USB 케이블** — 유선 연결용 (길이와 유형은 확정 예정)
3. **빠른 시작 가이드** — 설정 및 사용 안내

*참고: 최종 패키지에는 추가 액세서리가 포함될 수 있습니다. 제품 디자인이 확정되면 이 페이지를 업데이트하겠습니다.*
