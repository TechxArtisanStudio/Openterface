# 使用例 | Openterface KeyMod

KeyModは、迅速なキーボードとマウス入力が必要だが、フルキーボードが利用できないか不便な状況向けに設計されています。スマートフォンを入力デバイスとして使用—KeyModをターゲットに接続して制御を開始します。

## 主な使用例

### **屋外ディスプレイコンピュータ**

キーボードを持ち歩かずに屋外設置のディスプレイコンピュータを制御。KeyModはスマートフォンをディスプレイPCに接続し、設定、トラブルシューティング、コンテンツ更新を行います。

### **LED看板プレーヤー**

現場でLED看板プレーヤーとコントローラーを管理。KeyModを接続し、スマートフォンをキーボードとして使用して、キーボードを探すことなく変更を行います。

### **キオスク**

キオスクの設定、デバッグ、メンテナンスを迅速に。KeyModはキオスクOSやソフトウェアと対話する必要があるときに、即座にキーボードとマウスアクセスを提供します。

### **スマートTVとセットトップボックス**

ディスプレイはあるがリモコンでの操作が面倒なスマートTV、セットトップボックス、メディアプレーヤーを制御。スマートフォンをフルキーボードとトラックパッドとして使用します。

### **迅速なワークフローショートカット**

デバイス間の切り替えよりも速いワークフローが必要な場合、KeyModでスマートフォンを迅速な入力に使用できます。開発者、ITスタッフ、追加の機器を持ち歩かずに時々キーボードアクセスが必要な方に最適です。

## なぜKeyMod？

- **ポータブル** — スマートフォンは常に携帯；KeyModはコンパクト
- **プラグアンドプレイ** — ターゲットデバイスにソフトウェア不要
- **デュアル接続** — 信頼性のためのUSB、ケーブル不要のためのBluetooth
- **オープンソース** — プロファイル、マクロ、ワークフローをカスタマイズ
