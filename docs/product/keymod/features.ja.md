# **機能と仕様** | Openterface KeyMod

## プレローンチ状況

KeyModは現在プレローンチ開発中です。ハードウェアとソフトウェアを改良しています。[製品ページ](/product/keymod/)で進捗とローンチ通知を受け取れます。

> **注意：** 機能、仕様、デザインは開発の継続に伴い変更される可能性があります。

## 製品バリアント

- **2-in-1コネクターバージョン** — ノートパソコン、タブレット、スマートフォンとの幅広い互換性のためのUSB A + USB C複合プラグ設計
- **USB Cバージョン** — モダンデバイスとクリーンなケーブル管理のための専用USB Cプラグ

## コア機能

### **スマートフォンをキーボードとトラックパッドに**

KeyModはスマートフォンをポータブルキーボードとトラックパッドコンソールに変えます。フルキーボードとマウスが利用できない場合、またはデバイス切り替えよりも速いワークフローが必要な場合に使用します。屋外ディスプレイコンピュータ、LED看板プレーヤー、キオスク、スマートTV、セットトップボックスに最適です。

### **デュアル接続：USB + Bluetooth**

- **USB** — 信頼性が高く低レイテンシの入力のためのプラグアンドプレイ有線接続
- **Bluetooth** — シナリオが許容する場合のケーブル不要セットアップ；セットアップを軽量でポータブルに

KeyModはリモートデスクトップの代替ではなく、実用的な日常のローカルデバイス制御のために設計されています。

### **オープンソースモバイルアプリ**

オープンソースモバイルアプリで以下が可能です：

- カスタム入力プロファイルの作成
- マクロの起動とホットキーの設定
- ワークフローショートカットの作成

KeyModコントローラーアプリは**Android、IOS、iPadOS**に焦点を当てています。Openterfaceエコシステム全体では**WindowsとmacOS**のソフトウェアによるデスクトップコントロールの拡張も進めています。

### **真のハードウェアHID**

Openterface Mini-KVMの実績あるHIDコアを基盤としています。ハードウェアベースのキーボードとマウスエミュレーション—ターゲットデバイスにソフトウェアのインストールは不要です。

### **オープンソース**

KeyModはオープンハードウェアおよびオープンソースソフトウェアです。プロジェクトの進展に伴い、回路図、PCBファイル、ファームウェア、ソフトウェア、BOMを公開します。[コミュニティに参加](/discord)して貢献し、最新情報を入手してください。

## 技術仕様

### **接続**

- **USB**：USB Cポート（2-in-1バリアント：USB A + USB C複合プラグ）
- **Bluetooth**：HIDキーボードとマウス
- **ターゲット**：ソフトウェアのインストールは不要

### **主要ハードウェア**

- CH9329 HIDエミュレーターチップ
- Bluetoothモジュール
- USB Cポート
- MCU
- プログラマブルボタン

### **入力機能**

- フルキーボードとマウスエミュレーション（HID）
- カスタム入力プロファイル
- マクロとホットキー
