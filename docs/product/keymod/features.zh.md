# **功能与规格** | Openterface KeyMod

## 预发布状态

KeyMod 目前处于预发布开发阶段。我们正在完善硬件和软件。在[产品页面](/product/keymod/)订阅以了解最新进展和发布通知。

> **注意：** 功能、规格和设计在开发过程中仍可能发生变化。

## 产品版本

- **2 合 1 连接器版本** — 组合 USB A + USB C 插头设计，广泛兼容笔记本电脑、平板电脑和手机
- **USB C 版本** — 专用于现代设备的 USB C 插头，便于整洁的线缆管理

## 核心功能

### **手机变键盘和触控板**

KeyMod 将您的手机变成便携式键盘和触控板控制台。在无法使用完整键盘和鼠标时使用，或当您需要比切换设备更快的操作流程时使用。适用于户外显示屏电脑、LED signage 播放器、信息亭、智能电视和机顶盒。

### **双连接：USB + 蓝牙**

- **USB** — 即插即用有线连接，实现可靠、低延迟的输入
- **蓝牙** — 在场景允许时实现无线设置；保持轻便便携的配置

KeyMod 专为实用的日常本地设备控制而设计，而非替代远程桌面。

### **开源移动应用**

通过我们的开源移动应用，您可以：

- 创建自定义输入配置
- 启动宏并配置快捷键
- 扩展至高级模式：数字键盘和手柄
- 创建工作流快捷方式

KeyMod 控制器应用主要支持 **Android 和 iPadOS**。我们也在更广泛的 Openterface 生态系统中通过 **Windows 和 macOS** 软件扩展桌面控制。

### **虚拟游戏控制器**

KeyMod 可作为虚拟游戏控制器，支持熟悉的布局：

- **Xbox 布局**
- **PlayStation 布局**
- **NES 布局**

### **可编程硬件按钮**

KeyMod 包含可编程硬件按钮，用于设备上的操作，如快速触发和简单的宏式工作流。此功能仍处于实验阶段，将通过原型设计、测试和社区反馈不断完善。

### **真正的硬件 HID**

基于 Openterface Mini-KVM 的成熟 HID 核心。基于硬件的键盘和鼠标模拟——目标设备无需安装软件。

### **开源**

KeyMod 是开源硬件和开源软件。随着项目的发展，我们将发布电路图、PCB 文件、固件、软件和 BOM。[加入我们的社区](/discord)参与贡献并获取最新信息。

## 技术规格

### **连接**

- **USB**：USB C 端口（2 合 1 版本：USB A + USB C 组合插头）
- **蓝牙**：HID 键盘和鼠标
- **目标**：无需安装软件

### **关键硬件**

- CH9329 HID 模拟器芯片
- 蓝牙模块
- USB C 端口
- MCU
- 可编程按钮

### **输入功能**

- 完整键盘和鼠标模拟（HID）
- 自定义输入配置
- 宏和快捷键
- 数字键盘和手柄模式
- 虚拟游戏控制器布局（Xbox、PlayStation、NES）
