## Allgemein

**:material-chat-question:{ .faq } Was ist KeyMod?**

KeyMod ist ein kompakter USB + Bluetooth HID-Emulator (Tastatur und Maus), der Ihr Smartphone in eine tragbare Tastatur und ein Trackpad verwandelt. Verwenden Sie es zur Steuerung von Geräten, die Tastatur-/Mauseingaben benötigen – Kioske, Smart-TVs, Set-Top-Boxen, Außendisplays – ohne eine vollständige Tastatur mitzuführen.

**:material-chat-question:{ .faq } Welche Plattformen unterstützt die KeyMod-App?**

Die KeyMod-Controller-App konzentriert sich auf **Android** und **iPadOS**. Wir erweitern auch die Desktop-Steuerung mit **Windows- und macOS-Software** in unserem breiteren Openterface-Ökosystem.

**:material-chat-question:{ .faq } Benötigt das Zielgerät eine Software?**

Nein. KeyMod emuliert eine Standard-USB-Tastatur und -Maus. Das Zielgerät erkennt es als Plug-and-Play-HID-Hardware – keine Treiber oder Softwareinstallation erforderlich.

**:material-chat-question:{ .faq } Ist KeyMod Open Source?**

Ja. KeyMod ist Open-Hardware und Open-Source-Software. Wir werden Schaltpläne, PCB-Dateien, Firmware, Software und BOM veröffentlichen, während das Projekt sich weiterentwickelt.

## Konnektivität

**:material-chat-question:{ .faq } USB vs. Bluetooth – was soll ich verwenden?**

- **USB**: Zuverlässiger, geringere Latenz. Verwenden Sie es, wenn Sie die stabilste Verbindung wünschen.
- **Bluetooth**: Kabellos. Verwenden Sie es, wenn Sie ein leichteres Setup wünschen und das Szenario drahtlose Verbindung zulässt.

**:material-chat-question:{ .faq } Welche Hardware-Varianten sind geplant?**

1. **2-in-1-Stecker** — Kombinierter USB-A- und USB-C-Stecker für breite Kompatibilität
2. **USB-C-Version** — Dedizierter USB-C-Stecker für moderne Geräte

## Funktionen

**:material-chat-question:{ .faq } Welche Gamepad-Layouts werden unterstützt?**

KeyMod kann als virtueller Game-Controller mit **Xbox-Layout**, **PlayStation-Layout** und **NES-Layout** fungieren.

**:material-chat-question:{ .faq } Kann ich benutzerdefinierte Profile und Makros erstellen?**

Ja. Die Open-Source-Mobil-App unterstützt benutzerdefinierte Eingabeprofile, Makros, Hotkeys und Workflow-Verknüpfungen. Sie können auch Tastenfeld- und Gamepad-Modi verwenden.

**:material-chat-question:{ .faq } Was sind die programmierbaren Hardware-Tasten?**

KeyMod enthält programmierbare Hardware-Tasten für On-Device-Aktionen wie Schnellauslöser und einfache Makro-Workflows. Diese Funktion ist noch experimentell und wird durch Community-Feedback verfeinert.

## Pre-Launch

**:material-chat-question:{ .faq } Wann wird KeyMod auf den Markt kommen?**

KeyMod befindet sich in der Pre-Launch-Entwicklung. Abonnieren Sie die [Produktseite](/product/keymod/) für Launch-Benachrichtigungen und Updates.

**:material-chat-question:{ .faq } Wie hängt KeyMod mit Mini-KVM und KVM-Go zusammen?**

KeyMod nutzt den bewährten HID-Kern von Openterface Mini-KVM. Es teilt denselben hardwarebasierten Ansatz zur Tastatur- und Mausemulation, ist aber für einen anderen Anwendungsfall konzipiert: Ihr Smartphone in eine tragbare Tastatur/Trackpad für die lokale Gerätesteuerung zu verwandeln, anstatt KVM-over-USB-Videoaufnahme.
